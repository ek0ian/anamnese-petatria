# Anamnese Pet'Atria

Sistema web para gerenciamento de fichas de anamnese veterinária, baseado no modelo da clínica Camila Rosa Medicina Veterinária. Trabalho Prático Semestral de Arquitetura de Aplicações Web 2026.1.

## Visão geral

Aplicação web que permite que um médico veterinário cadastre pacientes (animais e seus tutores), preencha fichas de anamnese, registre vacinas, exames, atestados, prescrições e demais procedimentos clínicos.

## Stack

- Backend: .NET 10 (C#) — Web API
- Banco: MongoDB (rodando em Docker)
- Frontend: HTML + CSS + JavaScript (fetch / navegação assíncrona)
- Documentação: Swagger / OpenAPI
- Autenticação: JWT com RBAC (perfis `admin` e `veterinario`)
